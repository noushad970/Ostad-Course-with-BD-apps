# 📱 Flutter Multi-Screen UI Application

A clean and responsive **Flutter mobile application** demonstrating a basic multi-screen workflow with authentication UI, navigation, and settings management.

---

## ✨ Overview

This project showcases a simple but well-structured Flutter application with three core screens:

- 🔐 Login & Registration Screen  
- 🏠 Home Dashboard Screen  
- ⚙️ Settings Screen (Logout functionality)

It is designed for beginners to understand **Flutter UI structure, navigation, and screen separation**.

---

## 🎯 Key Objectives

- Build multi-screen Flutter applications
- Implement navigation between pages
- Design clean and reusable UI components
- Understand basic app flow (Login → Home → Settings → Logout)

---

## 🧱 Tech Stack

- **Framework:** Flutter  
- **Language:** Dart  
- **UI Toolkit:** Material Design
